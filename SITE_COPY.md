## 📋 Notas para Revisão Editorial

### Consistência Linguística
O site usa predominantemente **inglês** para copy editorial, com algumas exceções em **português**:
- Breadcrumbs usam "Inicio" (pt) em vez de "Home"
- Label de categoria usa "Categoria" (pt)
- Formulário de newsletter usa "Nome", "Inscrever", "Enviando..." (pt)
- Meta descriptions de busca estão em português
- HTML lang está definido como `pt-BR`

### Decisão Pendente
> ⚠️ **Recomendação:** Alinhar a língua para uma experiência coerente — ou todo em inglês (público global) ou todo em português (público lusófono). A mistura atual pode prejudicar a percepção profissional.

### Dados Dinâmicos
Os seguintes valores são preenchidos pela API e não são copy estática:
- `{post.title}` — Título do artigo
- `{post.excerpt}` — Resumo do artigo
- `{post.author.name}` — Nome do autor
- `{post.published_at}` — Data de publicação
- `{post.reading_time}` — Tempo de leitura estimado
- `{post.category.name}` — Nome da categoria
- `{tag.name}` — Nome da tag
- `{category.description}` — Descrição da categoria
- `{N}` — Contagem de resultados de busca
- `{query}` — Termo de busca inserido pelo utilizador
