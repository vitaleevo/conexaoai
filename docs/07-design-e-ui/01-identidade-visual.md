# Identidade Visual e Design System

O Conexão AI utiliza uma linguagem visual "Tech-Premium", focada em clareza, modernidade e foco no conteúdo.

## 1. Paleta de Cores (Design Tokens)
- **Primary**: Indigo/Violet (#6366f1) - Representa inteligência e inovação.
- **Background**: Dark Mode Slate (#0f172a) - Para reduzir a fadiga ocular e dar um ar profissional.
- **Surface**: Glassmorphism (transparências sutis) para criar profundidade.

## 2. Tipografia
- **Títulos**: 'Inter' ou 'Outfit' (Sans-serif moderna, peso 700+).
- **Corpo**: 'Inter' (peso 400) - Foco em legibilidade em telas de alta densidade.

## 3. Componentes (Shadcn UI)
Todos os componentes seguem os princípios de **Acessibilidade (WCAG)**.
- **Botões**: Estados claros de hover e foco.
- **Cards**: Bordas arredondadas suaves e sombras sutis.
- **Inputs**: Feedback visual imediato para erros e sucesso.

## 4. Voz e Tom
- **Técnico mas Acessível**: Explicamos conceitos complexos de forma que um iniciante entenda, mas sem subestimar o leitor experiente.
- **Direto ao Ponto**: Evitamos "encher linguiça". Cada parágrafo deve agregar valor.
